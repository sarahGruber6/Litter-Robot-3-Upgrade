My boyfriend and I got scammed off marketplace buying a broken Litter Robot 3. 
I figured I might as well upgrade it since all the sensors/motors work and it seemed fun.

In this repo I'm creating a replacement control board, as well as an upgrade board to the LED nightlight to have camera support, and replacing the weight switch with a proper load cell. 
The control board will be compatible with all OEM sensors while also having pins for upgraded boards/sensors. 

The end goal is to have this control board be accessible through local wifi to view sensor data and camera feed. I do not look forward to the software side of this. :-D
Feel free to tag along as I develop!
